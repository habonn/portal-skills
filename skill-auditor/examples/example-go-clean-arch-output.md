This project follows **Clean Architecture**


### Architecture Rules

#### Do's
- ✅ Keep business logic in domain layer
- ✅ Use dependency injection for external services
- ✅ Define interfaces in domain layer

#### Don'ts
- ❌ Import infrastructure code into domain layer
